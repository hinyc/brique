## 프로젝트 실행
node v20.9.0 / npm v10.6.0

### Sever
- npm install
- .env 환경변수 추가
(MariaDB 사용시 필요)
  - DB_HOST
  - DB_USERNAME
  - DB_PASSWORD
  - DB_NAME
- node app.js (server 폴더에서)

### Client
- npm install
- .env 환경변수 추가(Api host)
VITE_API_URL= http://localhost:3000
- npm run dev (cliernt 폴더에서)


### 프로젝트 환경
- React
- Typescript
- Emotion
- Vite
- Express
